# YouTube Clone

A YouTube Clone built with React and Vite that allows users to browse trending videos, search for videos, and watch them directly within the application.

## 🚀 Features

* Browse popular YouTube videos
* Search videos using the search bar
* Real-time search suggestions
* Watch videos directly in the app
* Dark mode support
* Responsive design
* Sidebar navigation
* Video details page

## 🛠️ Tech Stack

* React
* Vite
* JavaScript
* Tailwind CSS
* Redux Toolkit
* React Redux
* YouTube Data API v3

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Siddhantsin343/You-Tube-Clone.git
```

Move into the project directory:

```bash
cd You-Tube-Clone
```

Install dependencies:

```bash
npm install
```

Create a `.env` file in the root directory:

```env
VITE_GOOGLE_API_KEY={YOUR_API_KEY}
```

Start the development server:

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```
