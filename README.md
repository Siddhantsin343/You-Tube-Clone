# YouTube Clone

❗ A basic YouTube Clone built with React and Vite as a learning project to understand core frontend concepts such as React components, routing, state management with Redux Toolkit, API integration, search functionality, and responsive UI development.

This project is intended for educational purposes and focuses on implementing fundamental concepts rather than replicating every feature of YouTube.

## 🚀 Features

* Browse popular YouTube videos
* Search videos using the search bar
* Real-time search suggestions
* Watch videos directly in the app
* Dark mode support
* Responsive design
* Sidebar navigation
* Video details page

## 🛠️ Tech Stack

* React
* Vite
* JavaScript
* Tailwind CSS
* Redux Toolkit
* React Redux
* YouTube Data API v3

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Siddhantsin343/You-Tube-Clone.git
```

Move into the project directory:

```bash
cd You-Tube-Clone
```

Install dependencies:

```bash
npm install
```

Create a `.env` file in the root directory:

```env
VITE_GOOGLE_API_KEY={YOUR_API_KEY}
```

Start the development server:

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```
